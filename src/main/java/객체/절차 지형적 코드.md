
## 1. 절차 지향이란?
#### => 여러 프로시저가 데이터를 공유하는 것

#### 특징 & 문제점
1. 데이터를 공유하는 방식은 시간이 처음에는 간단하지만
   요구사항이 많아지면서 구조를 복잡하게 만들고 수정을 어렵게 만드는 요인이 된다.
   
2. 코드자체를 복사하게 되는 일이 많아진다. 


```java

    }

```
